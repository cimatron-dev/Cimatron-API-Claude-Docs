API Application (C#) Project

The steps required to create an API application are shown here. This example was created using Visual Studio 2022.

There are 5 steps;

  1. Create a new Solution
  2. Configure the Project
  3. Add Interop References
  4. Add the Manifest file
  5. Access the iApplication interface

Create a new Solution

The Solution acts as the container for all the work you will do.

  1. Start a New Project. Open Visual Studio and click File > New > Project (or use the keyboard shortcut Ctrl-Shift-N) to start the new project wizard.
  2. Filter the list of templates using the selectors at the top of the right hand pane. Use the filter settings C#, Windows, and Console.Â 
  3. In the list of results, select Console App (.NET Framework).
  4. Click Next.
  5. In the next wizard screen, provide a Project name, select a storage Location and select .NETÂ Framework 4.8 in the Framework dropdown. The Solution name is derived from the Project Name.

    NoteÂ If .NETÂ Framework 4.8 is not displayed, select any other framework in the dropdown,Â then follow the optional steps in Configure the Project.

  6. Click Create.

Add Interop References

Interop references provide specific functionality to the application, such as access to the commands and functions in Cimatron.

  1. If the Solution Explorer is not displayed, click View>Solution Explorer, or use the keyboard shortcut Ctrl+Alt+L.

  2. Right-click on References in Solution Explorer and select the Add Reference option.

  3. Browse to ..\Cimatron\ProgramÂ and add all Cimatron API Interop Files by selecting the checkboxes.

    Select the following files;
    • interop.CimAppAccess.dll
    • interop.CimatronE.dll
    • interop.CimBaseAPI.dll
    • interop.CimMdlrAPI.dll
    • interop.CIMMiscAPI.dll
    • interop.CimNcAPI.dll
    • interop.CimServicesAPI.dll
    • interop.CimElectrodeAPI.dll

Access the iApplication interface

There are two ways to access the interface, the one you use depends on whether your application will be used with a running instance of Cimatron, or if it will create a new Cimatron instance.

  1. Open Solution Explorer > Project Name > Program.csÂ and add the appropriate code in the static void Main(string[] args) section.

    • If Cimatron is running, theÂ AppAccess class gives access to the running application.Â 
      interop.CimAppAccess.AppAccess CimAppAcc = new interop.CimAppAccess.AppAccess();
      interop.CimatronE.IApplication CimApp = (interop.CimatronE.IApplication)CimAppAcc.GetApplication();
      var id = CimApp.ProcessId;

    • If Cimatron isnt running, open it - CimUtilityApp will open a new instance of Cimatron.
      interop.CimAppAccess.CimUtilityApp cimUtilityApp = new interop.CimAppAccess.CimUtilityApp();
      interop.CimatronE.IApplication CimApp = (interop.CimatronE.IApplication)cimUtilityApp.GetIApplication();
      int id = CimApp.ProcessId;
