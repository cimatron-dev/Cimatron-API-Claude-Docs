Notes:

  • The Plugin command can be created only with a managed language (VB.NET, C#).

  • To create the Plugin command, you need to load the following reference to the project:

    • cimuiinfrastructure.dll

  • There is no need to register the Plugin command.

  • The Plugin command can be a simple DLL and there is no need to register it.
