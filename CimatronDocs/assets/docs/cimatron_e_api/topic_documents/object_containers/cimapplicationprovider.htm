CimApplicationProvider

Use this interface to access the current Cimatron application.Â 

Namespace

interop.CimServicesAPI

Compatibility

Cimatron 2024 or later

Properties

None

Remarks

This interface can only be used on a Class project. See the sample use in Starting a C# Application with the Cimatron API.

Example

//Define a Cimatron Provider object
interop.CimServicesAPI.CimApplicationProvider aAppProvider =
new interop.CimServicesAPI.CimApplicationProvider();

import clr
#Adding references of required interop dlls
clr.AddReference("interop.CimAppAccess")
clr.AddReference("interop.CimatronE")
clr.AddReference("interop.CimServicesAPI")
clr.AddReference("interop.CimBaseAPI")
clr.AddReference("interop.CimMdlrAPI")

#importing modules from interop dlls
import interop.CimAppAccess
import interop.CimatronE
import interop.CimServicesAPI
import interop.CimBaseAPI
import interop.CimMdlrAPI

#Getting AppProvider Object
aCimAppProvider = System.Activator.CreateInstance(interop.CimServicesAPI.CimApplicationProviderClass)
