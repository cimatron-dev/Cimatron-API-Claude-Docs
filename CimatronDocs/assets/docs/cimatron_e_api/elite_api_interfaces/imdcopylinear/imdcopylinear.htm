IMdCopyLinear

This class represents a modeler procedure for a linear array copy of geometries.
Two types of linear copy are possible:

  1. by Copy Linear Mode Ucs. For this set : IMdCopyLinear::UcsEntity and other API's.
  2. by Copy Linear Mode Vector. For this set : IMdCopyLinear::DirectionVectorX, IMdCopyLinear::DirectionVectorY and other API's.
