Remarks

You can use it to check what kind of file you have. In MS Visual C++ if calling this interface with method QueryInterface(...)Â from file you get NULL, it means that this file isn't a part file. The same thing can be done in MS Visual BasicÂ Â using the Set method. Assigning a variable declared as IPartDocument type with the Set method a variable of the ICimDocument type will produce an error if there is no part file.
