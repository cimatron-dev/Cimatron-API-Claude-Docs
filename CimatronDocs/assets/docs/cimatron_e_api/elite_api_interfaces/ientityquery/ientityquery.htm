IEntityQuery

This interface is used to query a model entity for created procedures (like MdExtrude procedure) and specific entities, by a predefined filter or set of filters.

Example

interop.CimAppAccess.AppAccess AppAcc = new interop.CimAppAccess.AppAccess();
