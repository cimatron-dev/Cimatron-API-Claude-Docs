IMdScale::UniformMode

Use this property to get or set the mode either to Uniform or Non Uniform.

  • Uniform
    Mode = 1 : Enlarge or shrink geometry by a Uniform scale factor (that is, the same in all directions). The result of uniform scaling is similar (geometrically) to the original.
  • NonUniform
    Mode = 0 : Enlarge or shrink geometry by a Non-Uniform scale factor (with a separate scale factor for each axis direction). Non-uniform scaling occurs when at least one of the scaling factors is different from the others and changes the shape of the geometry; For example, a square may change into a rectangle.
