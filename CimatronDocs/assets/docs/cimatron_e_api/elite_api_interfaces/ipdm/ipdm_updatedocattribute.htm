IPdm::UpdateDocAttribute

Use this method to add a new attribute to a file or change data in an existing attribute.

Compatibility

Cimatron 2024 or later

Syntax

UpdateDocAttribute(string PathToFile, System.Object[] aArray, string aEmptyString, interop.CimatronE.AccessMode aMode);

Remarks

  • Any attributes in a file with the same names as in a called method will be changed with the new one. Additional new attributes will be attached to the file.
  • The parameter Owner temporary does not take affect. Set its value to "guest".
  • The parameter AccessMode temporary does not take affect.

Example

interop.CimAppAccess.AppAccess aAppAccess = new interop.CimAppAccess.AppAccess();
interop.CimatronE.IApplication aCimApp = (interop.CimatronE.IApplication)aAppAccess.GetApplication();
interop.CimatronE.IPdm aPdm = aCimApp.GetPdm();
//Get attributes from document
object oDocAtt = new object();
aPdm.RetriveDocAttribute(@"C:\PathToFile", ref oDocAtt, "");
object[] aDocAttArr = (object[])oDocAtt;
int count = aDocAttArr.Length;
interop.CimBaseAPI.IDocAttribute[] aArray = new interop.CimBaseAPI.IDocAttribute[count];
for (int i = 0; i< count; i++)
{
interop.CimBaseAPI.IDocAttribute aAttr = (interop.CimBaseAPI.IDocAttribute)aDocAttArr[i];
//change one of the attribute value
if (aAttr.Name == "TMPL::Heat_Treatment")
{
aAttr.Name = "MyCustomAttr";
aAttr.Value = "Its_Attached";
}
//push attribute object to array.
aArray[i] = aAttr;
}
//update the attributes of document
aPdm.UpdateDocAttribute(@"C:\PathToFile", aArray, "", interop.CimatronE.AccessMode.cmAccessModeWrite);
