INcDocument

This class represents one of document type in Cimatron(NcDocument).

In MS Visual C++, if calling this interface with method QueryInterface from file produces NULL, it means that this file isn't a part file. The same thing can be done in MS Visual BasicÂ Â using the Set method. Assigning a variable declared as IPartDocument type with the Set method a variable of the ICimDocument type will produce an error if there is no part file.
