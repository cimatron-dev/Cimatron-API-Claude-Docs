IPdmHooks::BeforeNewDocument

An event occurs when Cimatron creates a new file. In particular, this happens after selecting a file type and units of measure from the New File dialog.

This method passes you the full path, the type and the units of measure of a file to be created. The full path is a temporary path until the first save of the file.
