ICaptureImage

This interface represents an interaction interface of capture image. It helps to capture image of specified height and width.

// create Cimatron App Access object

interop.CimBaseAPI.IInteraction aInteraction = aInteractionSink.CreateInteraction(interop.CimBaseAPI.InteractionType.cmCaptureImage);
interop.CimServicesAPI.CaptureImage aCaptureImage = (interop.CimServicesAPI.CaptureImage)aInteraction;
