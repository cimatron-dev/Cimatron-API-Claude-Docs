IMdAxis

Use this interface to create an axis in four different ways:

  1. Parallel to an existing axis or straight curve, or two points.

  2. Intersection of two planar faces or planes.

  3. As normal to a plane or surface, axis or curve through a given point.

  4. Through a circular face or curve passing through its center.
