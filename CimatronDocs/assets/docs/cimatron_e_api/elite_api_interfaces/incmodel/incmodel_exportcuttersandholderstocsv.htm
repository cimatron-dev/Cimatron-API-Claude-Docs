INcModel::ExportCuttersAndHoldersToCSV

This function enables the export of Cutter, Holder and Material data from an open Cimatron NC file to CSV files.Â 

  • The function generates three files containing cutter data, holder data, and material data according to cutters selected.
  • If an empty list is submitted, the output will be all cutters in the current open file.

Compatibility

Cimatron 2024 or later

void ExportCuttersAndHoldersToCSV(string[] iElements, string iCuttersCSV, string iHoldersCSV, string iMaterialsCSV);
