INcModel::CreateFromTemplateReplaceCriteriaSets

The method creates NC toolpaths and procedures from the given template and replaces set names in criteria definitions according to the second input file, in all procedures created from the template. The template file name must have the suffix .tpt or .mtt (single or multi toolpath) but not .pct. The toolpaths and procedures are created in the active NC file.

Compatibility

Cimatron 2024 or later

Syntax
