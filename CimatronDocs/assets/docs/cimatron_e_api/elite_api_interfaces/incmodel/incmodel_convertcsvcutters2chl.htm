INcModel::ConvertCSVCutters2CHL

Use this function to convert a given CSV file to a CHL file and to the NC cutters paste buffer (A CHL file is an elt file that contains only cutter data, used mainly in cutter library).

Compatibility

Cimatron 2024 or later

Syntax

T_NcConvertCSV2CHLResult ConvertCSVCutters2CHL(string iSorceCSVFile, string iDestCHLFile, int iExportToCHL, int iExportToPasteBuffer, NcConflictMethod iConflictMethod);
