IMdCopySingle

This interface representsÂ a linearÂ copy ofÂ geometry.

Four types of linear copy are available:

  1. by two points.
    For this set : IMdCopySingle::RelocateMode,IMdCopySingle::OriginPoint,IMdCopySingle::DestinationPoint, IMdCopySingle::Entities

  2. by setting the delta parameter for X, Y, Z.
    For this set : IMdCopySingle::RelocateMode,IMdCopySingle::DeltaX,IMdCopySingle::DeltaY,IMdCopySingle::DeltaZ,IMdCopySingle::Entities

  3. by two UCS.
    For this set : IMdCopySingle::RelocateMode,IMdCopySingle::OriginUcs,IMdCopySingle::DestinationUcs,IMdCopySingle::Entities

  4. by vector and length of shift.
    For this set : IMdCopySingle::RelocateMode,IMdCopySingle::DirectionVector,IMdCopySingle::VectorDelta,IMdCopySingle::Entities
