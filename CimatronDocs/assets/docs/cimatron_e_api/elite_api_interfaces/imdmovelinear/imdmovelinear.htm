IMdMoveLinear

This interface represents a linear shift of geometry. Four types of linear move are possible:

  1. By two points.
    For this, set values for Entities,Mode,Ă‚Â OriginPosition, andĂ‚Â DestinationPosition.
  2. By setting the delta parameter for X,Y,Z.
    For this, set values for Entities, Mode, DeltaX, DeltaY, DeltaZ, RotationAroundXAxis, RotationAroundYAxis, and RotationAroundZAxis.
  3. By two UCS.
    For this, set values for Entities, Mode, OriginUcs, and DestinationUcs.
  4. By vector and length of shift.
    For this, set values for Entities, Mode, Direction, DirectionDelta, RotationAroundXAxis, RotationAroundYAxis, and RotationAroundZAxis.
