IMdCircle::CircleArcModes

Use this property to get and set various Circle Options for Arc/Circle creation.

  • cmThreePoints = Allows to create a Circle/Arc by setting 3 points

  • cmCenterAndRadius = Allows to create a Circle/Arc by setting its center point and defining its radius

  • cmTwoPointsAndRadius = Allows to create a Circle/Arc by setting 2 points and defining its radius

  • cmTangentToCurveAndPoint = Allows to create a Circle/Arc tangent to a 2D curve passing through a point

  • cmTangentToCurveAndTwoPoints = Allows to create a Circle/Arc tangent to a 2D curve passing through 2 points

  • cmTwoCurvesAndRadius = Allows to create a Circle/Arc with a defined radius tangent to two 2D curves

  • cmTwoCurvesAndPoint = Allows to create a Circle/Arc tangent to two 2D curves and passing through a point

  • cmThreeCurves = Allows to create a Circle/Arc tangent to three 2D curves
