ISetupValueData

The ISetupValueData Interface is used to update setup parameter values.

Namespace

interop.CimServicesAPI

Compatibility

Cimatron 2024 or later

Properties

Set GroupName string
Set ParamName string
Set Value double

Methods

None Â  Â 
Â  Â  Â 

Events

NoneÂ 

Remarks

This interface is set from ICimSetup:GetEmptySetupValueData().

This interface is used to update existing values only. You cannot create new parameters using this interface.

Example

interop.CimAppAccess.AppAccess AppAcc = new interop.CimAppAccess.AppAccess();
interop.CimatronE.IApplication CimApp = (interop.CimatronE.IApplication)AppAcc.GetApplication();
interop.CimatronE.ICimDocument aDoc = CimApp.GetActiveDoc();
interop.CimMdlrAPI.IModelContainer aContainer = (interop.CimMdlrAPI.IModelContainer)aDoc;
interop.CimMdlrAPI.IMdlrModel aMdlrModel = (interop.CimMdlrAPI.IMdlrModel)aContainer.Model;
interop.CimMdlrAPI.ICimSetup aCimSetup = aMdlrModel.GetCimSetup();
interop.CimMdlrAPI.ISetupValueData aSetupValueData = aCimSetup.GetEmptySetupValueData();
aSetupValueData.GroupName = "Group1";
aSetupValueData.ParamName = "Grp1Name";
aSetupValueData.Value = 111;
interop.CimMdlrAPI.ISetupValueData[] SetupNewValues = new interop.CimMdlrAPI.ISetupValueData[1];
SetupNewValues[0] = aSetupValueData;
aCimSetup.SetGeneralParamsValue(SetupNewValues);
