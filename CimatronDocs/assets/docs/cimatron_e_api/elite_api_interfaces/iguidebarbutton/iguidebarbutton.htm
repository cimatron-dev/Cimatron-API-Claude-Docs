IGuideBarButton

This interface adds a button to the guide bar. The IGuideBarButtonEventsDelegator interface captures the events from the CimatronE object (the GuideBarButton external tool); The connection-point object sends events from the XGuideBarButton class.

Namespace

interop.CimServicesAPI

Compatibility

Cimatron 2024 or later

Remarks

Initialize the Feature Guide variable with the command "WithEvents" to implement the events in VB.

Example

// create Cimatron App Access object
interop.CimAppAccess.AppAccess aAppAccess = new interop.CimAppAccess.AppAccess();
//Get application object from App access
interop.CimatronE.IApplication aCimApp = (interop.CimatronE.IApplication)aAppAccess.GetApplication();
interop.CimServicesAPI.IInteractionSink aInteractionSink = (interop.CimServicesAPI.IInteractionSink)aCimApp;
interop.CimServicesAPI.IInteraction aGuideBarInteraction = aInteractionSink.CreateInteraction(interop.CimServicesAPI.InteractionType.cmGuideBar);
interop.CimServicesAPI.GuideBar aGuideBar = (interop.CimServicesAPI.GuideBar)aGuideBarInteraction;
interop.CimServicesAPI.GuideBarButton aBotton = aGuideBar.CreateButton();
aBotton.Id = 1001;
string aText = "Start";
aBotton.Text = aText;
string aToolTip = "Start Execution";
aBotton.Tooltip = aToolTip;
aBotton.Type = interop.CimServicesAPI.GuideBarBtnType.cmGBFather;
aGuideBar.Create("MyGuidBar");
aGuideBar.AddButton(aBotton);
aGuideBar.Activate();
aGuideBar.Show();
