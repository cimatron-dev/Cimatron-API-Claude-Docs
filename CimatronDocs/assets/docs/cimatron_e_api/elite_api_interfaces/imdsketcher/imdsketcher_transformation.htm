IMdSketcher::Transformation

Use this property to get and set the transformation of a sketch default plane.

Compatibility

This feature is under development.

Syntax

Property get:

TransformationMatrix = Transformation ( );

Property set:

Transformation( TransformationMatrix );

Â 

Return Type

Return: (Variant) TransformationMatrix

Variant that contains a double type one-dimensional array of 12 elements that consist of an affine transformation matrix and a translation vector. First 9 elements are a transformation matrix, each 3 elements form one row in the matrix. Last 3 elements form a translation vector.
