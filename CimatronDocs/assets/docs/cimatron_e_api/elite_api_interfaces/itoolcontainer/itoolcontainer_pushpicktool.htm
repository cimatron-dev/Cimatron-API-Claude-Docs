IToolContainer::PushPickTool

This method sets a filter on a pick tool (it is an extension of the PushTool method).

The following parameters are required:

  • iActiveFilter (the actual filter)
  • iEnableFilter (the entities types that can be filtered)
  • iEnablePoints - if the filter includes points

Compatibility

Cimatron 2024 or later

Syntax

PushPickTool( ToolImplementedObject , iActiveFilter, iEnableFilter, iEnablePoints);
