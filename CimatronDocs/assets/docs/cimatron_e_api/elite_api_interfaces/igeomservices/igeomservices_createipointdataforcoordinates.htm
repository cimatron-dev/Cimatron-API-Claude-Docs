IGeomServices::CreateIPointDataForCoordinates

Use this method to create IPoint data from the X, Y, Z Coordinates.

Compatibility

Cimatron 2026 or later

Syntax

interop.CimServicesAPI.IPointData CreateIPointDataForCoordinates(double xCoordinate, double yCoordinate, double zCoordinate);

def CreateIPointDataForCoordinates(double, double, double) -> interop.CimServicesAPI.IPointData
