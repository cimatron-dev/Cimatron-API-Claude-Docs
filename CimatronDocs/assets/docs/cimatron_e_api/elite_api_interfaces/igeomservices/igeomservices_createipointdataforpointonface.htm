IGeomServices::CreateIPointDataForPointOnFace

Use this method to create IPoint data which will be used to give point on face. IPoint data will take care associativity.

Compatibility

Cimatron 2026 or later

Syntax

interop.CimServicesAPI.IPointData CreateIPointDataForPointOnFace(interop.CimServicesAPI.ICimEntity iEnt, double aUParam, double aVParam);

def CreateIPointDataForPointOnFace(iEnt : interop.CimServicesAPI.ICimEntity, double, double) -> interop.CimServicesAPI.IPointData

Return Type

Return: interop.CimServicesAPI.IPointData

Example

// create Cimatron App Access object
interop.CimAppAccess.AppAccess aAppAcc = new interop.CimAppAccess.AppAccess();
//Get application object from App access
interop.CimatronE.IApplication aCimApp = (interop.CimatronE.IApplication)aAppAcc.GetApplication();
// Get active document from application
interop.CimatronE.ICimDocument aDoc = aCimApp.GetActiveDoc();
interop.CimMdlrAPI.IModelContainer aContainer = (interop.CimMdlrAPI.IModelContainer)aDoc;
//GetModel
interop.CimMdlrAPI.IMdlrModel aMdlrModel = (interop.CimMdlrAPI.IMdlrModel)aContainer.Model;
//Get GeomService Object
interop.CimServicesAPI.IGeomServices aGeomService = (interop.CimServicesAPI.IGeomServices)aMdlrModel.GetGeomServicesObj();
interop.CimBaseAPI.IModel aModel = (interop.CimBaseAPI.IModel)aMdlrModel;
//Get Curve Using Id
interop.CimBaseAPI.ICimEntity aEnt = aModel.GetEntityById(12, aModel);
double aUParam = 0.10;
double aVParam = 0.20;
interop.CimServicesAPI.IPointData aPointData = aGeomService.CreateIPointDataForPointOnCurve((interop.CimServicesAPI.ICimEntity)aEnt, aUParam, aVParam, interop.CimServicesAPI.EPointType.cmPtClose2Face);
