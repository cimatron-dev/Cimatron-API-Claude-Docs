IFilterSet

Use multiple filters joined in the set. You can use saved sets, calling them from the file by name. For example, you can aggregate some simple filter typesÂ in one set for the first work session with a file, and use this saved set for selectionÂ in another work session without having to aggregate the compound filter again.

Remarks

To create a set use the ISetsFactory::CreateSet method.

Example

// create Cimatron App Access object
interop.CimAppAccess.AppAccess aAppAccess = new interop.CimAppAccess.AppAccess();
//Get application object from App access
interop.CimatronE.IApplication aCimApp = (interop.CimatronE.IApplication)aAppAccess.GetApplication();
interop.CimBaseAPI.ICimDocument aDoc = (interop.CimBaseAPI.ICimDocument)aCimApp.GetActiveDoc();
interop.CimMdlrAPI.IModelContainer aContainer = (interop.CimMdlrAPI.IModelContainer)aDoc;
interop.CimBaseAPI.IModel TmpModel = (interop.CimBaseAPI.IModel)aContainer.Model;
interop.CimMdlrAPI.IEntityQuery aEntityQuery = (interop.CimMdlrAPI.IEntityQuery)TmpModel;
interop.CimMdlrAPI.IEntityFilter aEntityFilter = aEntityQuery.CreateFilter(interop.CimMdlrAPI.EFilterEnumType.cmFilterSet);
interop.CimBaseAPI.FilterSet aFilterSet = (interop.CimBaseAPI.FilterSet)aEntityFilter;
aFilterSet.Add("Curves");

interop.CimBaseAPI.ISetsFactory SetFac = TmpModel.GetSetsFactory();
interop.CimBaseAPI.ISet aSet = SetFac.CreateSet("iSetsName", (interop.CimBaseAPI.IEntityFilter)aFilterSet);

int[] aFilterList = (int[])aFilterSet.GetFilter();

import clr
#Adding references of required interop dlls
clr.AddReference("interop.CimAppAccess")

#importing modules from interop dlls
