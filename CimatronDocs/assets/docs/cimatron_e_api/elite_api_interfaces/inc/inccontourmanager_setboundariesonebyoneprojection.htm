INcContourManager::SetBoundariesOneByOneProjection

Use this method to set the contour using one by one projection method. All the contours are projected in the plane of first entity in the input list.

Compatibility

Cimatron 2024SP3 or later

Syntax

void SetBoundariesOneByOneProjection(interop.CimNcAPI.ICimEntityList iList, double iMaxGap, interop.CimNcAPI.CreationMode iCreationMode, interop.CimNcAPI.SimplifyMode iSimplifyMode, double iMaxSmoothAngle);

interop.CimAppAccess.AppAccess AppAcc = new interop.CimAppAccess.AppAccess();
interop.CimatronE.IApplication CimApp = (interop.CimatronE.IApplication)AppAcc.GetApplication();
interop.CimatronE.ICimDocument aDOC = CimApp.GetActiveDoc();
interop.CimNcAPI.IModelContainer aModelCont = (interop.CimNcAPI.IModelContainer)aDOC;
