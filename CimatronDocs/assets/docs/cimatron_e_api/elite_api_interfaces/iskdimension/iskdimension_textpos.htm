Remarks

The point that defines the text position is located in the center of the lower line of the dimension text box. The type of dimension created depends on the dimension text position. For example, in the linear dimension, you can get the horizontal or vertical dimension; in the radial dimension, it is possible to get the outer or inner angle between the two curves.
