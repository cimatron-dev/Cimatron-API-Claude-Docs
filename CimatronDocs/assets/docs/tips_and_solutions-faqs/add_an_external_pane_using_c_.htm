Add an External Pane Using C# (.NET)

This document is a guide for adding an external pane inÂ Cimatron using the API. Perform the following steps to create an API application. The steps and images are based on Visual Studio 2022.

  1. Create a C# Class Library Project

    In Visual Studio, create a new C# project with the project type Class Library.

  2. Edit Project Properties

    1. Output path - Ensure the output path of this project is set to the Cimatron installation folder.

    2. Modify the .csprojÂ file
      Add the following <PropertyGroup> node in the .csproj file. This prevents the creation of a .net8.0-windows directory inside the output folder and directs the DLL output to the specified path.

      <PropertyGroup Condition="'$(Configuration)|$(Platform)'=='Debug|x64'">
      <AppendTargetFrameworkToOutputPath>false</AppendTargetFrameworkToOutputPath>
      <OutputPath>C:\Cimatron\dev\Elite\common\output\Debug_x64\</OutputPath>
      </PropertyGroup>

    3. Set Target Framework

      Change

      <TargetFramework>net8.0</TargetFramework>

      to

      <TargetFramework>net8.0-windows</TargetFramework>

      <UseWindowsForms>True</UseWindowsForms>

      <EnableComHosting>true</EnableComHosting>

  3. Add References to the Project

    Go to Dependencies > Add Project Reference and add the following:
