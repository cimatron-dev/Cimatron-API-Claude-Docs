Internal Plug-in Command

This section describes the process used to create an internal plug-in command in Cimatron. Internal plug-in and plug-in commands are similar, internal plug-in commands load the commands from the InternalPlugInCommandlist.xml while the normal plug-in commands load the commands from the ExternalCommands.ini file

Step 1. Export the Visual Studio Template

  1. Open the Cimatron Control Panel.

  2. Click the Visual Studio Templates option

  3. Select Project Type: Plugin Command and export the template.

Step2. Create a Visual Studio Project for the Plugin Command

  1. Open Visual Studio.

  2. Create a new project.

  3. Choose Cimatron C# Plugin Command as the project type.Â 

  4. Click Next. The project will be created with all required dependencies and references.

Step 3: Implement the Plugin Command

Plugin Command Class

The class (such as MyPluginCommand) is derived from ICimApiCommandPlugin.

For internal commands we support Name, Caption, Tooltip, Description, and ExecuteCommand. You can set these parameters in the existing code,Â or you can use following sample code.

public class MyPluginCommand : CimUIInfrastructure.PlugIn.ICimApiCommandPlugin

{

public CimUIInfrastructure.PlugIn.ApiCommand AppendCommand()

{

CimUIInfrastructure.PlugIn.ApiCommand Command1 = new CimUIInfrastructure.PlugIn.ApiCommand();

// Set Command name

Command1.Name = "My Command";

// Set Caption string

Command1.Caption = "My Command Caption";

// Set Tool tip string

Command1.ToolTip = "My Command Tool tip";

// Set Description string

Command1.Description = "My Command Tool Description";

// Call the execute class

Command1.ExecuteCommand = new MyCommandExecution();

// Return command object

return Command1;

}

}

Step 4: Build the project.

Ensure the build location is same as the Cimatron installation location. Then build the project.

Step 5. Add Command in the InternalPlugInCommandlist.xml

Open the InternalPlugInCommandlist file from theÂ Cimatron installation location.

In the file, add the plug-in command class name assigned to your command.

Example

<Command CimCommandID="52000" Customization="True" Keyboard="True" PlugInDll="CimatronAPICSharpPluginCommand1.MyPluginCommand" Customization_Tooltip="Valid Command" Application="Part" ExcludeApplication="None" />
