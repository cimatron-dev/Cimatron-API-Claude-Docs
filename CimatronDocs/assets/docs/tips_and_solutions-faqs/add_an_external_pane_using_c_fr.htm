Add an External Pane Using C# (.NET Framework)

This document is a guide for adding an external pane inÂ Cimatron using the API. Perform the following steps to create an API application. The steps and images are based on Visual Studio 2022.

  1. Create a C# Class Library Project

    In Visual Studio, create a new C# project with the project type Class Library.

  2. Edit Project Properties

    1. Output path - Ensure the output path of this project is set to the Cimatron installation folder.

    2. Edit Properties > AssemblyInfo.cs file to setÂ ComVisible true.

    3. Mark the checkboxÂ Register for COM interop.

  3. Add References to the Project.
    Go to References > Add Reference and add interop.CimserviceAPI.dll.

  4. Add a User Control form to the project. This form will be visible as a pane in Cimatron.

  6. Create an External Pane Class

    1. Add a new class file and insert the following code, updating the GUIDs as necessary.

      public class ExternalPaneClass: interop.CimServicesAPI.IExternalPane
      {
      public string GetControl()
      {
      return "PaneTesting.Pane";
      }
      public string GetPaneName()
      {
      return "TopBottom";
      }
      public InitialPanePosition GetInitialPanePosition()
      {
      return InitialPanePosition.cmBottomPane;
      }
      public int GetInitialWidth()
      {
      return 1000;
      }
      public int GetInitialHeight()
      {
      return 250;
      }
      public ApplicationMode GetApplicationMode()
      {
      return ApplicationMode.Part;
      }
      }

  7. Build the project in administration mode.

  8. Configure ExternalCommands.ini

    1. Open the ExternalCommands.ini file in a text editor from the ProgramData folder.Â 

    2. Under the [External Pane] add the following entry using the syntaxÂ "Namespace.ClassName=Namespace.ClassName@1"

      Example:

      CimatronAPIExternalPane.ExternalPaneClass=CimatronAPIExternalPane.ExternalPaneClass@1Â 

      Note: The class name and namespace mentioned here are from the class that is derived from interop.CimServicesAPI.IExternalPane.

    Use the Reset All command from theÂ Cimatron Control Panel to load your changes.
