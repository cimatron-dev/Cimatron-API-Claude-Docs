Restricting Access to Features using theÂ LicServ.dll file

The Cimatron API is capable of fetching the License details. This can be useful for a developer when building a utility that uses the Cimatron API, allowing the developer to grant access to features that use the Cimatron API to specific customers. There are two main ways to do this:

  1. The Cimatron License information can be fetched using the ICimLicense interface. Details about the license interface and available functionality are documented in the ICimLicense section.
  2. The LicServ.dll file can get additional information, such as customer id, modules, and so on. This capability is featured in this section.

Functions in LicServ.dll
